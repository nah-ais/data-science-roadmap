# 🚀 Data Science Journey 2026: Roadmap to Top 10 Global Tech

Repository ini dibuat untuk mendokumentasikan perjalanan belajar saya selama 12 bulan menuju posisi Data Scientist/Analyst di perusahaan global. Progres ini mencakup teori matematika, pemrograman, hingga deployment sistem AI.

## 📊 Status Saat Ini
- **Fase Aktif:** Fase 1 - Fondasi Data.
- **Progres Keseluruhan:** 🟦⬜⬜⬜⬜⬜⬜⬜⬜⬜ 10%
- **Target Kuartal Ini:** Menguasai Statistik Inferensial dan SQL Advanced.

## 🛠️ Tech Stack & Skills
- **Bahasa:** Python (Pandas, NumPy), SQL, R.
- **ML/DL:** Scikit-Learn, PyTorch, BERT/IndoBERT.
- **Visualisasi:** Power BI, Streamlit.
- **Engineering:** Git, FastAPI, Docker, MLOps.

## 📂 Struktur Repositori
- `roadmap/`: Kurikulum detail dan daftar sumber daya belajar.
- `logs/`: Jurnal mingguan mengenai apa yang dipelajari dan kendala teknis.
- `notebooks/`: File `.ipynb` berisi latihan kode dan proyek eksplorasi.
- `projects/`: Proyek skala besar (end-to-end).
